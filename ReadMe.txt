To use, follow these directions:

1. Put the file into your web server's root directory
2. Open your terminal then redirect to your web server's root directory
3. Make the bash file executable by typing "sudo chmod -R u+x laravel-auto-installer.sh"
4. Run the bash file by typing ". ./laravel-auto-installer.sh"

Note: 

Remove the quotes when typing the commands

if you choose Composer, you must first install composer on your system by "sudo apt-get install composer"
